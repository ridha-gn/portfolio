# Ridha Gnounou — DevOps & Cloud Security Portfolio

A modern, professional portfolio website built with vanilla HTML, CSS, and JavaScript.

## Features
- 🎨 Premium dark theme with glassmorphism and gradient accents
- ✨ Smooth scroll-reveal animations
- 📱 Fully responsive (mobile, tablet, desktop)
- 🏆 Achievement showcase (La Nuit de l'Info awards)
- 🛡️ Projects section with detailed case studies
- 📜 Certifications gallery
- 📬 Contact section with direct links

## Deploy to GitHub Pages
1. Push this folder to a GitHub repository
2. Go to Settings → Pages → Source: Deploy from branch → `main`
3. Your site will be live at `https://your-username.github.io/portfolio/`

## Technologies
- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Inter, JetBrains Mono)
- No frameworks or build tools required
